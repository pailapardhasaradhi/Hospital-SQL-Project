
# Hospital Management System – SQL Project

## Overview
This project demonstrates a complete SQL-based Hospital Management System, designed to manage patients, doctors, appointments, and billing information.

## Features
- Patient management
- Doctor management
- Appointment handling
- Billing system
- Views, Joins, Procedures, Triggers

## Files
- hospital_db.sql – Full SQL script
- README.md – Documentation

## Technologies
- SQL
- MySQL / MariaDB

## How to Use
1. Import `hospital_db.sql` into MySQL Workbench or phpMyAdmin.
2. Run the script to create the database and tables.
3. Execute queries, procedures, and triggers as required.
